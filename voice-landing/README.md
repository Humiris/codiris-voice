# Codiris Voice Landing Page

Modern Next.js landing page for Codiris Voice - the Voice OS for the next generation of computing.

## Getting Started

### Install dependencies
```bash
npm install
```

### Run development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.

### Build for production
```bash
npm run build
npm run start
```

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Radix UI Components
- Framer Motion

## Project Structure

- `/src/app` - Next.js app directory with pages
- `/src/components` - React components
- `/public` - Static assets

## Deployment

This landing page can be deployed on Vercel, Netlify, or any platform that supports Next.js.
